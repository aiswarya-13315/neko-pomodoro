# 🐱 neko pomodoro
### * stay focused, little one *

A kitty-themed pixel art Pomodoro timer built with Python + Tkinter. Cherry blossom pink theme, an animated black pixel cat that changes mood based on your session, and a glowing ring countdown timer.

## ✨ features

- 🐈‍⬛ **animated pixel black cat** — changes mood depending on your session
  - focus time → sitting upright, tail wagging
  - short break → sleeping with floating z z z
  - long break → happy with a floating ♥
- 💗 **glowing ring timer** with a smooth countdown arc
- 🌸 **cherry blossom pink theme** throughout
- ♥ **session tracker** — 4 heart dots that fill in as you complete focus sessions
- 🔔 **auto mode switching** — jumps to break when focus ends, back to work after break
- 🕹️ **pixel-style buttons** with a press-down effect — START/PAUSE, RESET, SKIP
- 🪶 **zero dependencies** — pure Python standard library, no pip needed!

## 🚀 getting started

### 1. clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/neko-pomodoro.git
cd neko-pomodoro
```

### 2. run it
```bash
python pomodoro.py
```

> **Linux users:** if tkinter is missing:
> ```bash
> sudo apt-get install python3-tk
> ```

no API key, no pip install — just run! 🎉

---

## 🍅 how pomodoro works

The Pomodoro technique is a productivity method:

1. focus for **25 minutes** 🐱
2. take a **5 minute** short break 😴
3. repeat 4 times
4. take a **15 minute** long break 🥳

neko pomodoro handles all the switching automatically!
